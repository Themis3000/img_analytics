Image analytics

Get analytics on a page simply by embedding an invisible image

Where can I use this?

Anywhere you can embed an image! This could be anything from a form post to a github profile. You can get data on the approximate amount of unique viewers, the total amount of hits, the approximate location of visitors, and even the flow users take between pages if you are able to embed on multiple linked pages (useful for something like viewing which repos people view from your profile on github)

How does it work?

Every time a webpage is loaded, it needs to also request each image on the page. In this case, it's making a request to our server for the image. When we get an image request we are able to log the requesting ip address in order to distinguish between users and get an approximate location on where the request is coming from. We then send the site back a 1x1 transparent image so that the tracker image is an invisible as possible.

The ip address of any users will never be visible to you, we do this in order to help protect user privacy
